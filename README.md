# A Simple Chess App ♟️

A lightweight, fully **offline** chess application for Android.  
Play **Player vs. AI** (against the computer) or **Player vs. Player** (hotseat mode on the same device) — no internet connection required.


## Features

- Completely **offline** — play anywhere, anytime
- Two game modes:
  - **Player vs. AI** (simple computer opponent)
  - **Player vs. Player** (two players on one device)
- Intuitive touch controls for moving pieces
- Full standard chess rules: checkmate, stalemate, castling, en passant, pawn promotion
- Clean, minimalistic user interface
- No ads, no in-app purchases, no tracking
